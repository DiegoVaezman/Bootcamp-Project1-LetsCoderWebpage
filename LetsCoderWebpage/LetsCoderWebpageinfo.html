Conceptos básicos para iniciarte

  • HTML

    <!--...-->
    Define un comentario
    <!DOCTYPE> 
    Define el tipo de documento
    <a>
    Define un hipervínculo
    <area>
    Define un área dentro de un mapa de imágenes
    <article>
    Define un artículo
    <aside>
    Define el contenido aparte del contenido de la página
    <b>
    Define texto en negrita
    <body>
    Define el cuerpo del documento
    <br>
    Define un salto de línea único
    <button>
    Define un botón en el que se puede hacer clic
    <div>
    Define una sección en un documento
    <dl>
    Define una lista de descripción
    <dt>
    Define un término / nombre en una lista de descripción
    <footer>
    Define un pie de página para un documento o sección.
    <form>
    Define un formulario HTML para la entrada del usuario
    <h1> to <h6>
    Define encabezados HTML
    <head>
    Contiene metadatos / información del documento.
    <header>
    Define un encabezado para un documento o sección
    <img>
    Define una imagen
    <input>
    Define un control de entrada
    <li>
    Define un elemento de lista
    <link>
    Define la relación entre un documento y un recurso externo
    <main>
    Especifica el contenido principal de un documento.
    <ol>
    Define una lista ordenada
    <p>
    Define un párrafo
    <strong>
    Define texto importante
    <style>
    Define información de estilo para un documento
    <table>
    Define una tabla
    <tbody>
    Agrupa el contenido del cuerpo en una tabla
    <td>
    Define una celda en una tabla
    <tfoot>
    Agrupa el contenido del pie de página en una tabla
    <th>
    Define una celda de encabezado en una tabla
    <thead>
    Agrupa el contenido del encabezado en una tabla
    <title>
    Define un título para el documento
    <tr>
    Define una fila en una tabla
    <ul>
    Define una lista desordenada
